# hadss/turbo-trans-core 通信传输工具组件

## 简介

高性能序列化框架核心库，需配合 turbo-trans-json 使用

## 详细介绍

### 简介
TurboTransCore 为高性能序列化解决方案核心库
需配合 TurboTransJSON 使用

### 依赖系统版本
SDK: API 12 Release 及以上

### 安装依赖
使用 ohpm 安装
深色代码主题复制# 安装序列化框架核心库
```bash
ohpm install @hadss/turbo-trans-core
```

### Sample 参考
开发者可以参考使用 TurboTransSample 的代码

### 贡献代码
使用过程中发现任何问题都可以提 Issue，当然，也非常欢迎发 PullRequest 共建。
